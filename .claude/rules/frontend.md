# Frontend Rules

- Use pnpm, never npm
- Svelte 5 runes syntax (`$state`, `$derived`, `$effect`)
- Tailwind CSS 4
- TypeScript strict mode
- Components in `src/lib/components/`
- Routes follow SvelteKit conventions
- Department pages use dynamic route `/dept/[id]`
