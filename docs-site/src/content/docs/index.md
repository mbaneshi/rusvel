# RUSVEL

> One binary, one human, infinite leverage.

RUSVEL is an AI-powered virtual agency built with Rust + SvelteKit. It gives a solo builder the leverage of an entire agency through 12 autonomous departments, each powered by AI agents.

## Quick Start

```bash
git clone https://github.com/mbaneshi/all-in-one-rusvel
cd all-in-one-rusvel
cargo run
# Open http://localhost:3000
```

## What's Inside

- **12 Departments** — Forge, Code, Content, Harvest, GTM, Finance, Product, Growth, Distribution, Legal, Support, Infra
- **God Agent** — Your AI companion that knows your identity, products, and mission
- **Knowledge/RAG** — fastembed + lancedb for semantic search over your documents
- **Self-Improvement** — The app can analyze and improve its own codebase
- **27 Rust crates** — Hexagonal architecture with 13 port traits
