
## Overview

The Infra department handles infrastructure and operations -- CI/CD pipelines, deployment management, service monitoring, and incident response. It ensures your services are running, healthy, and cost-efficient.

## Quick Actions

| Action | What It Does |
|--------|-------------|
| **Deploy status** | Show deployment status across services and environments |
| **Health check** | Run health checks on all monitored services |
| **Incident report** | Show open incidents with severity and resolution status |

## Example Prompts

- "Show the current deployment status across all environments."
- "Run health checks on all services and report any issues."
- "Create an incident report for the API outage at 14:00 UTC."
- "What is our monthly infrastructure cost breakdown?"
- "Set up a monitoring alert for API response times above 500ms."

## Capabilities

- **Deploy** -- CI/CD pipeline management and deployment tracking
- **Monitor** -- service health monitoring and alerting
- **Incident** -- incident response tracking with severity levels and timelines
